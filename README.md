# My_First_Portfolio

This repository contains the deliverables for the **ALX: Introduction to HTML** project.

## Structure
- `html_basic/`
  - `base_index.html` — minimal valid HTML (Task 0)
  - `index.html` — first webpage (headings, 4+ paragraphs, clickable image) + unified semantic structure + learners table (Tasks 1, 3, 4)
  - `tweets.html` — embedded single tweet + unified structure and navigation (Tasks 2, 3)
- `portfolio/`
  - `index.html` — Personal Portfolio page using semantic HTML5 (Task 5)

## Requirements
- No external libraries (no Bootstrap/React/Vue/NodeJS).
- Code should be **W3C compliant**. Validate by upload: https://validator.w3.org/#validate_by_upload
- Files end with a newline.

## How to view
Open any `.html` file directly in your browser.  
(Optional) Enable GitHub Pages: **Settings → Pages → Deploy from a branch → main**.
